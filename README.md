# 🎀 Happy Birthday Amelie

Романтичный поздравительный сайт для Амели на её 18-летие!

## ✨ Возможности

- 💕 Романтичный дизайн с мягкими розовыми тонами
- 🎵 Встроенный плеер для музыки (добавь "suga-suga.mp3")
- 📸 Галерея для фото (нажми на фото, чтобы добавить свои)
- ✨ Красивые анимации и падающие сердца
- 📱 Адаптивный дизайн для всех устройств
- 💌 Секция с пожеланиями

## 🚀 Как использовать

1. **Добавь музыку:**
   - Положи файл `suga-suga.mp3` в корневую папку проекта

2. **Добавь фото:**
   - Нажимай на серые квадраты в галерее
   - Выбирай свои фото со своего компьютера
   - Фото будут отображаться в галерее

3. **Открой сайт:**
   - Просто открой `index.html` в браузере
   - Или задеплой на GitHub Pages

## 📂 Структура файлов

```
happy-birthday/
├── index.html     # Главный HTML файл
├── style.css      # Стили и оформление
├── script.js      # Интерактивность и анимации
├── suga-suga.mp3  # Музыка (добавь сам)
└── README.md      # Этот файл
```

## 💻 Как задеплоить на GitHub Pages

1. Перейди в **Settings** репозитория
2. Найди **Pages**
3. Выбери **Deploy from a branch**
4. Выбери **main** ветку
5. Нажми **Save**
6. Твой сайт будет доступен по ссылке: `https://imrannasirov00-dev.github.io/happy-birthday`

## 🎨 Как изменить цвета

Открой `style.css` и найди эти цвета:
- `#ff69b4` - основной розовый
- `#ff1493` - тёмный розовый
- `#ffe4f0` - светлый розовый
- Замени на свои цвета

## 🎵 Добавь музыку

1. Скачай песню "Suga Suga" в формате MP3
2. Переименуй в `suga-suga.mp3`
3. Положи в корневую папку проекта (рядом с index.html)

## 💕 Изменение текста

Открой `index.html` и измени:
- Имя Амели
- Возраст (18)
- Пожелания и сообщения
